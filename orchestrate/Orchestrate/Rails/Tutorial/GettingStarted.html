    Getting started with orchestrate-rails¶ ↑

    Orchestrate.io offers a free developer tier that makes it easy to start integrating DBaaS into your rails applications.

    Signing up for Orchestrate.io¶ ↑

    1. Go to orchestrate.io to set up an account.

    2. Check out the orchestrate.io docs and tutorial.

    3. Create an application using the orchestrate.io dashboard.

    4. You should now have a newly generated api-key for basic authentication.

    Creating the configuration file¶ ↑

    Create the file, orchestrate_config.json, used to set up the orchestrate.io client. Replace the value of user with the api-key that was generated when you created the films application:

    {
  "base_url":"https://api.orchestrate.io/v0",
  "user":"put-api-key-from-orchestrate.io-here",
  "verbose":"true"
}

    Setting verbose to true prints each HTTP request line to the standard output.
