    module Orchestrate::Rails

    orchestrate-rails¶ ↑

    Ruby gem orchestrate-rails maps rails models to the orchestrate.io api. The rails interface to orchestrate.io collections is defined in the Model class, which is intended to provide ActiveRecord-style functionality.

    Try out the orchestrate-rails tutorial!

    Usage example¶ ↑

    Define the orchestrate-rails model in <rails-root>/app/models/film.rb

    class Film < Orchestrate::Rails::Model
end

    By default, it is expected that the collection name can be derived from the class name as shown:

    • class: Film => 'films'

    • class: FilmClassic => 'film_classics'

    Collection names that do not follow this convention must be specified by adding the :define_collection_name key to the params hash during model initialization.

    class Film < Orchestrate::Rails::Model

  def initialize(params={})
    params[:define_collection_name] = "Classic_Film_Collection"
    super params
  end
end

    Define controller actions in <rails-root>/app/controllers/films_controller.rb

    class FilmsController < ApplicationController

  def index
    @films = Film.all
  end

  def new
    @film = Film.new
  end

  def create
    @film = Film.create! params[:film].merge(id: id)
  end

  def show
    @film = Film.find(params[:id])
  end
end
